KJXQZ
=====

`KJXQZ`_ is an Apache2 licensed collection of `anagram word solver`_ tools.

.. _`anagram word solver`: http://www.kjxqz.com/

.. image:: https://api.travis-ci.org/grantjenks/kjxqz.com.svg?branch=master
   :target: http://www.grantjenks.com/docs/kjxqz/

.. image:: https://ci.appveyor.com/api/projects/status/github/grantjenks/kjxqz.com?branch=master&svg=true
   :target: http://www.grantjenks.com/docs/kjxqz/

Quickstart
----------

Installing `KJXQZ`_ is simple with `pip <https://pypi.org/project/pip/>`_::

    $ pip install kjxqz

You can access documentation in the interpreter with Python's built-in `help`
function. The `help` works on modules, classes and methods in `KJXQZ`_.

.. code-block:: python

    >>> import kjxqz
    >>> help(kjxqz)
